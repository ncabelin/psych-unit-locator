About

Psych Unit Locator is a tool for calculating distances between L.A. County mental health facilities and a given location and provide useful info about each facilities. It also provides ease of search information via a filter searchbar.


A user can input a zipcode or use geolocation coordinates from the user device's built in gps. This web app uses the Google maps Places API for querying geographic coordinates for the zipcode via a NodeJS backend service and an algorithm for measuring the distance between two geographic coordinates.


An authorized user can edit, add, delete and update the database through the login system

INSTRUCTIONS

  1. To check nearest psych unit locations from a device, go to menu SEARCH then choose BY GPS.
  2. To check psych unit nearest to a zipcode, click BY ZIPCODE, then type zipcode in the input field zipcode and click SUBMIT.
  3. To search and filter info put your searchword in the input box labeled search.
  4. Click menu CONTACTS for contact info.
  5. If you are an authorized user you can edit the database by accessing the menu option LOGIN.
